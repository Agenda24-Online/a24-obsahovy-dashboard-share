# WordPress 7.1 a moderní vývoj pluginů

## Shrnutí
WordPress se posouvá směrem ke spolupráci v editoru, Reactu 19 a modernějšímu zpracování médií. U vlastních pluginů je důležité nepsat zastaralý editorový JavaScript.

## Doporučení
Do technického zadání pluginů přidat větu: „JavaScriptová část pluginu musí být napsaná podle aktuálních WordPress/Gutenberg standardů, bez zastaralých React metod a bez závislosti na neveřejných interních API WordPressu.“
